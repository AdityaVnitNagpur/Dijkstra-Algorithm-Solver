Instructions

  • Click anywhere on the screen to create a node.
  • Click button "Add edges" for adding edges between nodes.
  • Edges can be added by clicking on one node and then on other node.
  • All edges have default weights scaled to the distance between them.
  • You can click on the default weight and change it according to your need.
  • Enter the source node in input box and click "Run" to find minimum cost paths from source node to every node.
  • Node 0 will be considered as source node in case no input is given to "Source Node" box.
  • Maximum number of nodes that can be created is equal to 12.
